## Prerequisites
Make sure the following are installed:
- Modern web browser (Chrome, Firefox, Edge, Safari)  
- Node.js and npm (optional, if you want to build or customize Tailwind CSS locally)  
- A code editor (VS Code recommended)  

## Quickstart Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/jrwnnnn/ecoaction.git
2. (Optional) Install Tailwind CSS dependencies if you want to rebuild styles:
   ```bash
   npm install
   npm run tailwind

## Bringing it to Life
Just open the `index.html` file in your browser.
